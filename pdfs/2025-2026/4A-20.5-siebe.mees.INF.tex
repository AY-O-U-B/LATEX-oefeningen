$ \displaystyle\int \cosh^3 x dx $\\

$ = \displaystyle\int \left(\dfrac{e^x+e^{-x}}{2}\right)^3 dx $\\

$ = \displaystyle\int \dfrac{\left(e^x+e^{-x}\right)^3}{8} dx $\\

$ = \dfrac{1}{8}\displaystyle\int \left(e^x+e^{-x}\right)^3 dx $\\

$ = \dfrac{1}{8}\displaystyle\int \left(e^{3x}+3e^x+3e^{-x}+e^{-3x}\right) dx $\\

$ = \dfrac{1}{8}\left(\displaystyle\int e^{3x} dx + \displaystyle\int 3e^x dx + \displaystyle\int 3e^{-x} dx + \displaystyle\int e^{-3x} dx\right) $\\

$ = \dfrac{1}{8}\left(\dfrac{1}{3}e^{3x}+3e^x-3e^{-x}-\dfrac{1}{3}e^{-3x}\right) + C $\\

$ = \dfrac{1}{8}\left(\dfrac{1}{3}\left(e^{3x}-e^{-3x}\right)+3\left(e^x-e^{-x}\right)\right) + C $\\

$ = \dfrac{1}{8}\left(\dfrac{1}{3}\cdot 2\sinh 3x + 3\cdot 2\sinh x\right) + C $\\

$ = \dfrac{1}{12}\sinh 3x + \dfrac{3}{4}\sinh x + C $\\
