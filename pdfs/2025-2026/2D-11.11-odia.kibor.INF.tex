$\lim\limits_{x \rightarrow 8} \dfrac{2 - \sqrt[3]{x}}{x-8}$\\

$(\Rightarrow a^3 - b^3 = (a-b)(a^2 + ab + b^2))$\\

$= \lim\limits_{x \rightarrow 8} \dfrac{(2 - \sqrt[3]{x})\cdot(4 + 2\sqrt[3]{x} + (\sqrt[3]{x})^2)}{(x-8)\cdot(4 + 2\sqrt[3]{x} + (\sqrt[3]{x})^2)}$\\

$= \lim\limits_{x \rightarrow 8} \dfrac{8-x}{(x-8)\cdot(4 + 2\sqrt[3]{x} + (\sqrt[3]{x})^2)}$\\

$= \lim\limits_{x \rightarrow 8} \dfrac{-\cancel{(x-8)}}{\cancel{(x-8)}\cdot(4 + 2\sqrt[3]{x} + (\sqrt[3]{x})^2)}$\\

$= \dfrac{-1}{12}$
