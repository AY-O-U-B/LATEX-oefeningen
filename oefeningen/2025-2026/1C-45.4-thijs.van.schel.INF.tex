$z^3 = 8 \Rightarrow z^3 = 8+0i$\\

$r = \sqrt{0^2+8^2} = 8$\\

$\tan(\theta) = \dfrac{0}{8} = 0 \Rightarrow \theta = \operatorname{Bgtan}(0) = \pi$\\

$z^3 = \sqrt[n]{r}\operatorname{cis}\left(\dfrac{\theta + 2k\pi}{n}\right)$\\

$\Rightarrow \sqrt[3]{8}\operatorname{cis}\left(\dfrac{0+2k\pi}{3}\right)$\\

$z_0 = 2\operatorname{cis}(0) = 2(\cos(0)+i\sin(0)) = 2(1+0i) = 2$\\

$z_1 = 2\operatorname{cis}\left(\dfrac{2\pi}{3}\right) = 2\left(\cos\left(\dfrac{2\pi}{3}\right) + i\sin\left(\dfrac{2\pi}{3}\right)\right)$\\

$ = 2\left(\dfrac{-1}{2}+\dfrac{\sqrt{3}}{2}i\right) = -1+i\sqrt{3}$\\

$z_2 = 2\operatorname{cis}\left(\dfrac{4\pi}{3}\right) = 2\left(\dfrac{-1}{2}+\left(\dfrac{-\sqrt{3}}{2}\right)i\right) = -1-i\sqrt{3}$
