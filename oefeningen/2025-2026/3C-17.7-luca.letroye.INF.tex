$ \lim\limits_{x \rightarrow 0} \dfrac{x\sin^2 x}{x\cos x - \sin x} = \dfrac{0}{0}$\\


$\overset{H}{=}  \lim\limits_{x \rightarrow 0} \dfrac{\sin^2 x + x(2\sin x\cos x)}{\cos x + x(-\sin x) - \cos x}  $\\

$= \lim\limits_{x \rightarrow 0} \dfrac{\sin^2 x + x\sin(2x)}{-x\sin x} = \dfrac{0}{0}  $\\

$= \lim\limits_{x \rightarrow 0} \dfrac{\sin^2 x + x(2\sin x \cos x)}{-x\sin x} = \lim\limits_{x \rightarrow 0} \dfrac{\sin x + 2x \cos x}{-x}  $\\

$\overset{H}{=} \lim\limits_{x \rightarrow 0} \dfrac{3\cos x -2x\sin x}{-1} = -3$\\
