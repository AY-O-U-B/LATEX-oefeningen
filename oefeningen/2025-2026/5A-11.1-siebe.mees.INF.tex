$ \displaystyle\int_{0}^{\tfrac{\pi}{2}} x^2 \sin x dx $\\

$ \left \{
\begin{array}{l}
u = x^2 \\
dv = \sin x dx
\end{array}
\right .
\Rightarrow
\left \{
\begin{array}{l}
du = 2x dx \\
v = -\cos x
\end{array}
\right . $\\

$ = \left[ -x^2 \cos x \right]_{0}^{\tfrac{\pi}{2}} + \displaystyle\int_{0}^{\tfrac{\pi}{2}} 2x \cos x dx $\\

$ = \left[ -x^2 \cos x \right]_{0}^{\tfrac{\pi}{2}} + 2\displaystyle\int_{0}^{\tfrac{\pi}{2}} x \cos x dx $\\

$ \left \{
\begin{array}{l}
u = x \\
dv = \cos x dx
\end{array}
\right .
\Rightarrow
\left \{
\begin{array}{l}
du = dx \\
v = \sin x
\end{array}
\right . $\\

$ = \left[ -x^2 \cos x \right]_{0}^{\tfrac{\pi}{2}} + 2\left( \left[ x \sin x \right]_{0}^{\tfrac{\pi}{2}} - \displaystyle\int_{0}^{\tfrac{\pi}{2}} \sin x dx \right) $\\

$ = \left[ -x^2 \cos x \right]_{0}^{\tfrac{\pi}{2}} + 2\left( \left[ x \sin x \right]_{0}^{\tfrac{\pi}{2}} - \left[ -\cos x \right]_{0}^{\tfrac{\pi}{2}} \right) $\\

$ = \left( -\dfrac{\pi^2}{4}\cos \dfrac{\pi}{2} - 0 \right) + 2\left( \dfrac{\pi}{2}\sin \dfrac{\pi}{2} - ( -\cos \dfrac{\pi}{2} - ( -\cos 0 ) ) \right) $\\

$ = 0 + 2\left( \dfrac{\pi}{2} - 1 \right) $\\

$ = \pi - 2 $\\
