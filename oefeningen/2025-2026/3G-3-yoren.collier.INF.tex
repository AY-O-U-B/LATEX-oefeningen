$y - y_1 = a (x - x_1) \asa y -2 = a (x-0) \asa y = ax +2$\\

De rico van de raaklijn in een punt wordt bepaald door de afgeleide in dat punt.\\

$\adR f'(0) = -3 \cdot 0^2 -1 = -1$\\

$\adR$ Er zijn 3 distincte gevallen om te bekijken. $a = -1$ of $a < -1$ of $a > -1$\\

Als $a = -1$:  $y = -1x +2$ en $y = 2 -x -x^3$ $\adR -x +2 = -x^3 -x +2 \asa -x^3 \schrap{-x} \schrapb{+2} \schrap{+x} \schrapb{-2} = 0 \asa -x^3 = 0 \asa x = 0$\\

Als $a < -1$:  $y = -2x +2$ en $y = 2 -x -x^3$ $\adR -2x +2 = -x^3 -x +2 \asa -x^3 -x \schrapb{+2} +2x \schrapb{-2} = 0 \asa -x^3 +x = 0 \asa x \Lh -x^2 +1 \Rh = 0 \asa x = 0$ of $-x^2 = -1 \asa x = \pm 1$\\

Als $a > -1$:  $y = 1x +2$ en $y = 2 -x -x^3$ $\adR x +2 = -x^3 -x +2 \asa -x^3 -x \schrapb{+2} -x \schrapb{-2} = 0 \asa -x^3 -2x = 0 \asa x \Lh -x^2 -2 \Rh = 0 \asa x = 0$ of $-x^2 = +2 \asa  x^2 = -2 \adR$ geen extra reële nulpunten\\

Wanneer $a \ge -1$ is er maar 1 snijpunt tussen de rechte en de functie. Wanneer $a < -1$ zijn er 3 snijpunten tussen de rechte en de grafiek.\\
