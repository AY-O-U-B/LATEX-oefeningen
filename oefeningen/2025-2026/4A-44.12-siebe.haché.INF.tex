$ \displaystyle\int\dfrac{9x^2 +x +4}{x^3-x^2+x-1}dx$\\

$= \displaystyle\int\dfrac{9x^2 +x +4}{(x-1)(x^2+1)}dx$\\

$\Rightarrow \dfrac{9x^2 +x +4}{(x-1)(x^2+1)} = \dfrac{Ax+B}{x^2+1}+ \dfrac{C}{x-1}$\\

$x = 1$:\\

$C = \dfrac{9(1)^2+1+4}{1^2+1} = \dfrac{14}{2} = 7$\\

$x = i$:\\

$Ai + B = \dfrac{9(i)^2+i+4}{i-1} = \dfrac{-5+i}{-1+i}$\\

$= \dfrac{(-5+i)(-1-i)}{(-1+i)(-1-i)} = \dfrac{5+5i-i+1}{2} = \dfrac{6+4i}{2} = 3 + 2i$\\

$\Rightarrow A = 2$ en $B = 3$.\\

$\Rightarrow \displaystyle\int\left(\dfrac{2x + 3}{x^2 +1} + \dfrac{7}{x-1}\right)dx$\\

$= \displaystyle\int\dfrac{2x}{x^2 +1} dx+ 3\displaystyle\int\dfrac{1}{x^2+1} dx+ 7\displaystyle\int\dfrac{1}{x-1} dx$\\

$= \ln(x^2+1) + 3 \Bgtan(x) + 7\ln\left|x-1\right|+ C$
