$\displaystyle\int_{-1}^{2} (x \cdot \lfloor x \rfloor) dx  $\\

$\left \{
\begin{array}{l} 
-1 \text{ voor } -1 \le x < 0 \\
0 \text{ voor } 0 \le x < 1 \\
1 \text{ voor } 1 \le x < 2
\end{array}
\right .
\Rightarrow
\left \{
\begin{array}{l} 
-x \text{ voor } -1 \le x < 0 \\
0 \text{ voor } 0 \le x < 1 \\
x \text{ voor } 1 \le x < 2
\end{array}
\right .$

$\displaystyle\int_{-1}^{2} (x \cdot \lfloor x \rfloor) dx  $\\

$=\displaystyle\int_{-1}^{0} -x dx + \displaystyle\int_{0}^{1} 0 dx + \displaystyle\int_{1}^{2} x dx$\\

$= \left[-\dfrac{x^2}{2} \right]_{-1}^{0} + 0 + \left[\dfrac{x^2}{2} \right]_{1}^{2}$\\

$= \dfrac{1}{2} + \dfrac{3}{2}$\\

$= 2$
