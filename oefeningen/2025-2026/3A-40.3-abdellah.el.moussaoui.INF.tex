$f(x)=x^4\cos x \ln x$\\

$ f'(x)=\left(x^4\cos x \ln x\right)'$\\

$= \left(x^4\cos x\right)'\ln x+x^4\cos x \left(\ln x\right)'$\\

$= \left( \left( x^4\right)'\cos x +  x^4 \left(\cos x\right)' \right)\ln x+x^4\cos x \left(\ln x\right)'$\\

$= \left( 4 x^3\cos x -  x^4 \sin x \right)\ln x+x^3\cos x $\\
