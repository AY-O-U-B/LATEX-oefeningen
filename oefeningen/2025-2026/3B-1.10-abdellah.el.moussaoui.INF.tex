$f(x)=\sin x \cos x$\\

$ f'(x)=\left(\sin x \cos x\right)'$\\

$= \left(\sin x\right)'\cos x + \sin x \left(\cos x\right)'$\\

$= \cos^2 x - \sin^2 x $\\

$ f''(x)=\left(\cos^2 x - \sin^2 x\right)'$\\

$ = -2 \cos x \sin x - 2 \sin x \cos x$\\

$ = -4 \sin x \cos x$
