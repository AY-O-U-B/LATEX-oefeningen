$\theta = \dfrac{2\pi}{3}$\\

$z = 2\left(\cos \left(\dfrac{\dfrac{2\pi}{3}+2k\pi}{4}\right) + i \sin \left(\dfrac{\dfrac{2\pi}{3}+2k\pi}{4}\right) \right) = 2\left(\cos \left(\dfrac{\pi}{6} + \dfrac{k\pi}{2}\right) + i \sin \left(\dfrac{\pi}{6} + \dfrac{k\pi}{2}\right)\right ) $\\

$z_0 = 2\left(\cos \left(\dfrac{\pi}{6} + \dfrac{0\cdot \pi}{2}\right) + i \sin \left(\dfrac{\pi}{6} + \dfrac{0 \cdot \pi}{2}\right)\right ) = 2\left(\dfrac{\sqrt{3}}{2} + i\dfrac{1}{2}\right ) = \sqrt{3} + i $\\

$z_1 = 2\left(\cos \left(\dfrac{\pi}{6} + \dfrac{1\cdot \pi}{2}\right) + i \sin \left(\dfrac{\pi}{6} + \dfrac{1 \cdot \pi}{2}\right)\right ) = -1+i\sqrt{3} $\\

$z_2 = 2\left(\cos \left(\dfrac{\pi}{6} + \dfrac{2\cdot \pi}{2}\right) + i \sin \left(\dfrac{\pi}{6} + \dfrac{2 \cdot \pi}{2}\right)\right ) = -\sqrt{3} - i $\\

$z_3 = 2\left(\cos \left(\dfrac{\pi}{6} + \dfrac{3\cdot \pi}{2}\right) + i \sin \left(\dfrac{\pi}{6} + \dfrac{3 \cdot \pi}{2}\right)\right ) = 1 -i\sqrt{3} $\\
