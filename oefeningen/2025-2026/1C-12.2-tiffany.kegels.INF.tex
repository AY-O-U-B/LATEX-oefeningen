$z = \dfrac{1+i}{\sqrt{2}}$\\

\textbf{Teller} = 1+i\\

$r_1 = \sqrt{1^2 + 1^2} = \sqrt{2}$\\

$\tan \alpha_1 = \dfrac{1}{1} = 1 \Rightarrow \alpha_1 = \dfrac{\pi}{4}$\\

$\Rightarrow \sqrt{2} \operatorname{cis} \dfrac{\pi}{4}$\\

\textbf{Noemer} = $\sqrt{2}$\\

$r_2 = \sqrt{0^2 + (\sqrt{2})^2} = \sqrt{2}$\\

$\tan \alpha_2 = 0 \Rightarrow \alpha_2 = 0$\\

$\Rightarrow \sqrt{2} \operatorname{cis} 0$\\

$z = \dfrac{\sqrt{2} \operatorname{cis} \dfrac{\pi}{4}}{\sqrt{2} \operatorname{cis} 0} = \operatorname{cis} \left( \dfrac{\pi}{4} - 0 \right)$\\

$=\operatorname{cis} \dfrac{\pi}{4}$\\
