$I = \displaystyle\dint_{-3}^0 \left(x^2+3x+2\right)dx$\\

$= \displaystyle\dint_{-3}^{-2} \left(x^2+3x+2\right)dx - \displaystyle\dint_{-2}^{-1} \left(x^2+3x+2\right)dx + \displaystyle\dint_{-1}^0 \left(x^2+3x+2\right)dx$\\

$= \left[\dfrac{x^3}{3} + \dfrac{3x^2}{2} + 2x\right]_{-3}^{-2} - \left[\dfrac{x^3}{3} + \dfrac{3x^2}{2} + 2x\right]_{-2}^{-1} + \left[\dfrac{x^3}{3} + \dfrac{3x^2}{2} + 2x\right]_{-1}^{0}$\\

$= \left(\dfrac{-8}{3} + 6 - 4\right) - \left(\dfrac{-27}{3} + \dfrac{27}{2} - 6\right) - \left[\left(\dfrac{-1}{3} + \dfrac{3}{2} - 2\right) - \left(\dfrac{-8}{3} + 6 - 4\right)\right]  - \left(\dfrac{-1}{3} + \dfrac{3}{2} - 2\right)$\\

$= \dfrac{5}{6} + \dfrac{1}{6} + \dfrac{5}{6} = \dfrac{11}{6}$\\
