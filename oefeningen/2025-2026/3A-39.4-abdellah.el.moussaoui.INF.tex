$f(x)=x^{\sin x}$\\

$\Rightarrow y=x^{\sin x}$\\

$\Rightarrow  \ln y=\ln x^{\sin x} = \left(\sin x\right) \ln x$\\

$\Rightarrow \left(\ln y\right)'= \left(\left(\sin x\right) \ln x \right)'$\\

$\Rightarrow \dfrac{y'}{y} = \left(\left(\cos x\right) \ln x\right) + \dfrac{\sin x}{x}$\\

$\Rightarrow y' = y \left(\left(\left(\cos x\right) \ln x\right) + \dfrac{\sin x}{x}\right)$\\

$\Rightarrow y' =  x^{\sin x}\left(\cos x\right) \ln x + x^{\sin x-1}\sin x$\\


$\Rightarrow \boxed{f'(x) = x^{\sin x}\left(\cos x\right) \ln x + x^{\sin x-1}\sin x}$\\
