$ \displaystyle\int \dfrac{\sin x dx}{(1 + 3\cos^2 x)\cos x} $\\

$u =\cos x$\\

$du = -\sin x dx$\\

$\Rightarrow - \displaystyle\int \dfrac{1}{(1 + 3u^2)u}du$\\

$= -\displaystyle\int \left(\dfrac{A}{u} + \dfrac{Bu + C}{1 + 3u^2}\right)du$\\
 
$ A =\dfrac{1}{1 + 3u^2}\bigg|_{u = 0} = 1 $\\

$1 + 3u^2 = 0$\\

$\Leftrightarrow 3u^2 = -1$\\

$\Leftrightarrow u^2 = -\dfrac{1}{3}$\\

$\Leftrightarrow u = \dfrac{i}{\sqrt{3}}$\\

$B\dfrac{i}{\sqrt{3}} + C =\dfrac{1}{u}\bigg|_{u = \frac{i}{\sqrt{3}}} = \dfrac{1}{\frac{i}{\sqrt{3}}}$\\

$B\dfrac{i}{\sqrt{3}} + C = \dfrac{\sqrt{3}}{i}$\\

$B\dfrac{i}{\sqrt{3}} + C = -\sqrt{3}i$\\

$\Rightarrow (B, C) = (-3, 0)$\\

$\Rightarrow -\displaystyle\int \left(\dfrac{1}{u} - \dfrac{3u}{1 + 3u^2}\right)du$\\

$= -\displaystyle\int \dfrac{1}{u}du + \displaystyle\int \dfrac{3u}{1 + 3u^2}du$\\

$= - \ln |u| + \displaystyle\int \dfrac{3u}{1 + 3u^2}du$\\

$v = 1 + 3u^2$\\

$dv = 6u du$\\

$\dfrac{dv}{2} = 3u du$\\

$\Rightarrow \displaystyle\int \dfrac{3u}{1 + 3u^2}du = \dfrac{1}{2}\displaystyle\int \dfrac{1}{v}dv$\\

$= \dfrac{1}{2}\ln |v|$\\

$= \dfrac{1}{2}\ln |1 + 3u^2|$\\

$\Rightarrow \displaystyle\int \dfrac{\sin x dx}{(1 + 3\cos^2 x)\cos x} = - \ln |u| + \dfrac{1}{2} \ln |1 + 3u^2| + C$\\

$= - \ln |\cos x| + \dfrac{1}{2} \ln |1 + 3\cos^2 x| + C$\\
