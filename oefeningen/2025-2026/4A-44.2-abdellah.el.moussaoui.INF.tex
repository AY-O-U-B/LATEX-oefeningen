$\displaystyle\int \dfrac{3x^2+x+4}{x^3+x^2+x+1}dx$\\

$x^3+x^2+x+1 = x^2 \left(x+1\right)+1 \left(x+1\right) = \left(x+1\right) \left(x^2+1\right)$\\

$\dfrac{3x^2+x+4}{\left(x+1\right) \left(x^2+1\right)} = \dfrac{A}{x+1} + \dfrac{Bx+C}{x^2+1}$\\

$A = \left. \dfrac{3x^2+x+4}{x^2+1}\right|_{x=-1} = 3$\\

$Bi+C = \left. \dfrac{3x^2+x+4}{x+1}\right|_{x=i} = \dfrac{3i^2+i+4}{i+1} = \dfrac{1+i}{1+i} = 1$\\

$\Rightarrow B = 0, C = 1$\\

$\Rightarrow \displaystyle\int \left(\dfrac{3}{x+1} + \dfrac{1}{x^2+1}\right)dx$\\

$= 3\displaystyle\int \dfrac{dx}{x+1} + \displaystyle\int \dfrac{dx}{x^2+1}$\\
