$ \displaystyle\int \dfrac{1}{x(x^2+1)}dx $ \\

$ \dfrac{1}{x(x^2+1)} = \dfrac{A}{x} + \dfrac{Bx + C}{(x^2+1)}$ \\

$ \Rightarrow A = \dfrac{1}{x^2 + 1} = 1 $ met $ x = 0$ \\

$ \Rightarrow Bi + C = \dfrac{1}{x} = \dfrac{1}{i}$ met  $ x = i$ \\

$ \Rightarrow Bi + C = \dfrac{1}{i} = \dfrac{i}{i \cdot i} = -i$ \\

$ \Rightarrow B = -1 $ en $ C = 0 $ \\

$ = \displaystyle\int \dfrac{1}{x(x^2+1)}dx = \displaystyle\int \dfrac{1}{x}dx + \displaystyle\int \dfrac{-x}{(x^2 + 1)}dx $ \\

$ = \ln|x| - \dfrac{1}{2}\ln|x^2 + 1| + c$ \\
