Opgave: Bereken de Taylorpolynoom\\

$f(x, y) = x^3 + x^2y^3-3y^4$ in $(1,-1)$, graad $2$\\

$f(1, -1) =  -3$\\

Eerste orde parti\"ele afgeleiden:\\

$\dfrac{\partial f}{\partial x} = 3x^2+2xy^3  \Rightarrow \dfrac{\partial f}{\partial x}(1, -1) = 1$\\

$\dfrac{\partial f}{\partial y} = 3x^2y^2-12y^3  \Rightarrow \dfrac{\partial f}{\partial y}(1, -1) = 15$\\

Tweede orde parti\"ele afgeleiden:\\

$\dfrac{\partial^2 f}{\partial x^2} = 6x+3y^3 \Rightarrow \dfrac{\partial^2 f}{\partial x^2}(1, -1) = 4$\\

$\dfrac{\partial^2 f}{\partial y^2} = 6x+3y^3 \Rightarrow \dfrac{\partial^2 f}{\partial y^2}(1, -1) = -42$\\

$\dfrac{\partial^2 f}{\partial x\partial y} = \dfrac{\partial^2 f}{\partial y\partial x} = 6xy^2 \Rightarrow \dfrac{\partial^2 f}{\partial x\partial y}(1, -1) = \dfrac{\partial^2 f}{\partial y\partial x}(1, -1) = 6$\\

Taylorpolynoom:\\

$T_2 (x, y) = -3 - (x-1) + 15(y+1) - \dfrac{1}{2!}\left(4(x-1)^2+12(x-1)(y+1)-42(y+1)^2\right)  $\\

$ = -14 +3x -33y +2x^2 -21y^2 + 6xy$\\
