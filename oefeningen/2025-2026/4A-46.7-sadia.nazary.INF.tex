$\dint\dfrac{x^5-x^4+4x+4}{x^4+4}\,dx$\\

We voeren eerst de euclidische deling uit:\\

$\dfrac{x^5-x^4+4x+4}{x^4+4}
=
x-1+\dfrac{8}{x^4+4}$\\

Dus:\\

$\dint\dfrac{x^5-x^4+4x+4}{x^4+4}\,dx
=
\dint(x-1)\,dx+\dint\dfrac{8}{x^4+4}\,dx$\\

$=
\dfrac{x^2}{2}-x+\dint\dfrac{8}{x^4+4}\,dx$\\

We ontbinden de noemer:\\

$x^4+4=x^4+4x^2+4-4x^2$\\

$=(x^2+2)^2-(2x)^2$\\

$=(x^2-2x+2)(x^2+2x+2)$\\

Dus:\\

$\dint\dfrac{8}{x^4+4}\,dx
=
\dint\dfrac{8}{(x^2-2x+2)(x^2+2x+2)}\,dx$\\

Volgens de regel van Fuss schrijven we:\\

$\dfrac{8}{(x^2-2x+2)(x^2+2x+2)}
=
\dfrac{Ax+B}{x^2-2x+2}
+
\dfrac{Cx+D}{x^2+2x+2}$\\

We bepalen nu $A,B,C,D$ met de regel van Fuss.\\

Voor $x^2+2x+2$ nemen we een nulpunt:\\

$x=-1+i$\\

Dan geldt:\\

$C(-1+i)+D
=
\left.
\dfrac{8}{x^2-2x+2}
\right|_{x=-1+i}$\\

$-C+Ci+D
=
\dfrac{8}{(-1+i)^2-2(-1+i)+2}$\\

$-C+Ci+D
=
\dfrac{8}{4-4i}$\\

$-C+Ci+D
=
\dfrac{2}{1-i}$\\

$-C+Ci+D
=
\dfrac{2(1+i)}{(1-i)(1+i)}$\\

$-C+Ci+D=1+i$\\

Hieruit volgt:\\

$C=1$\\

$-C+D=1$\\

$D=2$\\

Voor $x^2-2x+2$ nemen we een nulpunt:\\

$x=1+i$\\

Dan geldt:\\

$A(1+i)+B
=
\left.
\dfrac{8}{x^2+2x+2}
\right|_{x=1+i}$\\

$A+Ai+B
=
\dfrac{8}{(1+i)^2+2(1+i)+2}$\\

$A+Ai+B
=
\dfrac{8}{4+4i}$\\

$A+Ai+B
=
\dfrac{2}{1+i}$\\

$A+Ai+B
=
\dfrac{2(1-i)}{(1+i)(1-i)}$\\

$A+Ai+B=1-i$\\

Hieruit volgt:\\

$A=-1$\\

$A+B=1$\\

$B=2$\\

Dus:\\

$\dfrac{8}{(x^2-2x+2)(x^2+2x+2)}
=
\dfrac{-x+2}{x^2-2x+2}
+
\dfrac{x+2}{x^2+2x+2}$\\

Bijgevolg:\\

$\dint\dfrac{x^5-x^4+4x+4}{x^4+4}\,dx
=
\dfrac{x^2}{2}-x
+
\dint\dfrac{-x+2}{x^2-2x+2}\,dx
+
\dint\dfrac{x+2}{x^2+2x+2}\,dx$\\

We werken de eerste integraal uit:\\

$I_1=\dint\dfrac{-x+2}{x^2-2x+2}\,dx$\\

$=
-\dfrac12\dint\dfrac{2x-4}{x^2-2x+2}\,dx$\\

$=
-\dfrac12\dint\dfrac{2x-2-2}{x^2-2x+2}\,dx$\\

$=
-\dfrac12\dint\dfrac{2x-2}{x^2-2x+2}\,dx
+
\dint\dfrac{1}{x^2-2x+2}\,dx$\\

$=
-\dfrac12\ln(x^2-2x+2)
+
\dint\dfrac{1}{(x-1)^2+1}\,dx$\\

$=
-\dfrac12\ln(x^2-2x+2)
+
\Bgtan(x-1)$\\

We werken de tweede integraal uit:\\

$I_2=\dint\dfrac{x+2}{x^2+2x+2}\,dx$\\

$=
\dfrac12\dint\dfrac{2x+4}{x^2+2x+2}\,dx$\\

$=
\dfrac12\dint\dfrac{2x+2+2}{x^2+2x+2}\,dx$\\

$=
\dfrac12\dint\dfrac{2x+2}{x^2+2x+2}\,dx
+
\dint\dfrac{1}{x^2+2x+2}\,dx$\\

$=
\dfrac12\ln(x^2+2x+2)
+
\dint\dfrac{1}{(x+1)^2+1}\,dx$\\

$=
\dfrac12\ln(x^2+2x+2)
+
\Bgtan(x+1)$\\

Dus de totale primitieve is:\\

$\dint\dfrac{x^5-x^4+4x+4}{x^4+4}\,dx$\\

$=
\dfrac{x^2}{2}-x
-\dfrac12\ln(x^2-2x+2)
+\Bgtan(x-1)$\\

$+
\dfrac12\ln(x^2+2x+2)
+\Bgtan(x+1)
+C$\\
