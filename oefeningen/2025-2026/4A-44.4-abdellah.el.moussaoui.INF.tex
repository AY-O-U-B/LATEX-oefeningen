$\displaystyle\int \dfrac{x^4+x^3-3x+5}{x^3+2x^2+2x+1}dx$\\

$x^3+2x^2+2x+1 = \left(x+1\right)\left(x^2+x+1\right)$\\

$\dfrac{x^4+x^3-3x+5}{x^3+2x^2+2x+1} = x-1+\dfrac{6-2x}{x^3+2x^2+2x+1}$\\

$\dfrac{6-2x}{\left(x+1\right)\left(x^2+x+1\right)} = \dfrac{A}{x+1}+\dfrac{Bx+C}{x^2+x+1}$\\

$A = \left.\dfrac{6-2x}{x^2+x+1}\right|_{x = -1} = 8$\\

$Bx + C = \left.\dfrac{6-2x}{x+1}\right|_{x = -\tfrac{1}{2}+i\tfrac{\sqrt3}{2}}$\\

$ = \dfrac{7-i\sqrt3}{\dfrac{1}{2}+i\dfrac{\sqrt3}{2}}$\\

$ = \dfrac{2\left(7-i\sqrt3\right)}{1+i\sqrt3}$\\


$ = \dfrac{2\left(7-i\sqrt3\right)\left(1-i\sqrt3\right)}{\left(1+i\sqrt3\right)\left(1-i\sqrt3\right)}$\\

$ = \dfrac{2\left(4-8i\sqrt3\right)}{4}$\\

$ = 2-4i\sqrt3$\\


$\Rightarrow -\dfrac{B}{2}+C+i\dfrac{B\sqrt3}{2} = 2-4i\sqrt3$\\

$\Rightarrow \dfrac{B\sqrt3}{2}=-4\sqrt3 \Rightarrow B= -8$\\

$\Rightarrow -\dfrac{-8}{2}+C=2 \Rightarrow C = -2$\\

$ =  \displaystyle\int\left(x-1\right)dx + 8\displaystyle\int\dfrac{dx}{x+1} -\displaystyle\int\dfrac{8x+2}{x^2+x+1}dx$\\
