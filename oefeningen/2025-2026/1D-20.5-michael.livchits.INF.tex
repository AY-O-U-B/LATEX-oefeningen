$z^4 + 4 = 0$\\

$z^4 + 4 + 4z^2 - 4z^2 = 0$\\

$(z^2+2)^2 - 4z^2 = 0$\\

$(z^2+2)^2 - (2z)^2 = 0$\\

$(z^2+2+2z)(z^2+2-2z) = 0$\\

Eerst oplossen voor $(z^2+2+2z)$ :\\

$D = -4$\\

$\sqrt{D} = 2i$\\

$z_1 = \dfrac{-2+2i}{2} = -1 + i$\\

$z_2 = \dfrac{-2-2i}{2} = -1 - i$\\

Daarna oplossen voor $(z^2+2-2z)$ :\\

$D = -4$\\

$\sqrt{D} = 2i$\\

$z_1 = \dfrac{2+2i}{2} = 1 + i$\\

$z_2 = \dfrac{2-2i}{2} = 1 - i$\\

$\Rightarrow f(z) = (z^2+2+2z)(z^2+2-2z)$ \\
