$f(x)= 2 \sin x - \sin^2 x$\\

Periode: $2\pi \Rightarrow$ onderzoek $\left[-\pi, \pi \right]$\\

Domein: $\mathbb{R}$\\

Asymptoten: geen\\

Nulpunten $f(x)$:\\

$2\sin x - \sin^2 x = 0$\\

$\sin x (2-\sin x)=0$\\

$\sin x = 0 \Rightarrow x=-\pi,0,\pi$\\

\medskip

$f'(x)=2\cos x - 2\sin x \cos x$\\

$=2\cos x - \sin 2x$\\

Nulpunten $f'(x)$:\\

$2\cos x - 2\sin x \cos x=0$\\

$2\cos x(1-\sin x)=0$\\

$\cos x=0 \Rightarrow x=-\dfrac{\pi}{2},\dfrac{\pi}{2}$\\

$\sin x=1 \Rightarrow x=\dfrac{\pi}{2}$\\

$\Rightarrow x=-\dfrac{\pi}{2},\dfrac{\pi}{2}$\\

\medskip

$f''(x)=-2\sin x - 2\cos 2x$\\

Nulpunten $f''(x)$:\\

$-2\sin x - 2\cos 2x=0$\\

$\sin x + \cos 2x=0$\\

$\sin x + 1 - 2\sin^2 x=0$\\

$2\sin^2 x - \sin x -1=0$\\

$(2\sin x +1)(\sin x -1)=0$\\

$\sin x=-\dfrac{1}{2} \Rightarrow x=-\dfrac{5\pi}{6},-\dfrac{\pi}{6}$\\

$\sin x=1 \Rightarrow x=\dfrac{\pi}{2}$\\
