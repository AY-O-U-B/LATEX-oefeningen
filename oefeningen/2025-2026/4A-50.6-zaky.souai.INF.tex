$ \displaystyle\int \sin x \cos 3xdx $\\

$ = \displaystyle\int \dfrac{1}{2} \left(\sin (x + 3x) + \sin (x - 3x) \right) dx $\\

$ = \displaystyle\int \dfrac{1}{2} \left(\sin 4x + \sin (-2x) \right) dx $\\

$ = \displaystyle\int \dfrac{1}{2} \left(\sin 4x - \sin 2x \right) dx $\\

$ = \dfrac{1}{2} \displaystyle\int \left(\sin 4x - \sin 2x \right) dx $\\

$ = \dfrac{1}{2} \left(-\dfrac{1}{4} \cos 4x + \dfrac{1}{2} \cos 2x \right) + C $\\

$ = -\dfrac{1}{8} \cos 4x + \dfrac{1}{4} \cos 2x + C $
