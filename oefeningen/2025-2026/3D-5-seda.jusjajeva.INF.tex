$f(x) = \dfrac{3x^2 + ax + 6}{3x^2 + bx + 9}$ \\

$f'(x) = \dfrac{\dfrac{d}{dx}(3x^2 + ax + 6)\cdot(3x^2 + bx + 9) - (3x^2 + ax + 6)\cdot\dfrac{d}{dx}(3x^2 + bx + 9)}{(3x^2 + bx + 9)^2}$ \\

$= \dfrac{(6x + a)\cdot(3x^2 + bx + 9) - (3x^2 + ax + 6)\cdot(6x + b)}{(3x^2 + bx + 9)^2}$ \\

$= \dfrac{18x^3 + 6bx^2 + 54x + 3ax^2 + abx + 9a - 18x^3 - 6ax^2 - 36x - 3bx^2 - abx - 6b}{(3x^{2} + bx + 9)^2} $ \\

$= \dfrac{-3ax^2 + 3bx^2 + 18x + 9a - 6b}{(3x^2 + bx + 9)^2}$ \\

$f'(x)$ moet $0$ zijn als $x$ een extreme waarde is: \\

$\Leftrightarrow-3ax^2 + 3bx^2 + 18x + 9a - 6b = 0$ \\

$f'(1): -3a + 3b + 18 + 9a - 6b = 0$ \\

$\Leftrightarrow 2a - b = -6$ \\

$f'(2): -3a\cdot2^2 + 3b\cdot2^2 + 18\cdot2 + 9a - 6b = 0$ \\

$\Leftrightarrow 3a - 6b = 36$ \\
