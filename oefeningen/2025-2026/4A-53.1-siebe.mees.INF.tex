$ \displaystyle\int \dfrac{dx}{\sin^4 x} $\\

$ = \displaystyle\int \dfrac{1}{\sin^2 x} \cdot \dfrac{1}{\sin^2 x} dx $\\

$ = \displaystyle\int \dfrac{1}{\sin^2 x} \left( 1 + \cot^2 x \right) dx $\\

$ t = \cot x \left( dt = -\dfrac{1}{\sin^2 x} dx \right) $\\

$ = -\displaystyle\int \left( 1 + t^2 \right) dt $\\

$ = -\left( t + \dfrac{t^3}{3} \right) + C $\\

$ = -t - \dfrac{t^3}{3} + C $\\

$ \cot x = t $\\

$ \displaystyle\int \dfrac{dx}{\sin^4 x} = -\cot x - \dfrac{\cot^3 x}{3} + C $\\
