$ \displaystyle\int_{0}^{2\pi} \sin^2 \dfrac{x}{2} dx $\\

$ = \displaystyle\int_{0}^{2\pi} \dfrac{1 - \cos x}{2} dx $\\

$ = \dfrac{1}{2}\displaystyle\int_{0}^{2\pi} (1 - \cos x) dx $\\

$ = \dfrac{1}{2}\left( \displaystyle\int_{0}^{2\pi} 1dx - \displaystyle\int_{0}^{2\pi} \cos x dx \right) $\\

$ = \dfrac{1}{2}\left( \left[ x \right]_{0}^{2\pi} - \left[ \sin x \right]_{0}^{2\pi} \right) $\\

$ = \dfrac{1}{2}\left( (2\pi - 0) - (\sin(2\pi) - \sin(0)) \right) $\\

$ = \dfrac{1}{2}(2\pi - 0) $\\

$ = \pi $\\
