$x_n = -13$, $s_n = -45$, $v = -2$\\

Dit is een rekenkundige rij met formules:\\

$x_n = x_1 + (n-1)v$\\

$s_n = \dfrac{n(x_1 + x_n)}{2}$\\

Uit de eerste formule:\\

$x_1 = x_n - (n-1)v = -13 - (n-1)(-2) = -13 + 2(n-1)$\\

Invullen in de somformule:\\

$-45 = \dfrac{n\left(-13 + 2(n-1) + (-13)\right)}{2} = \dfrac{n(2n-28)}{2} = n(n-14)$\\

$n^2 - 14n + 45 = 0$\\

$(n-5)(n-9) = 0$\\

$n = 5$ of $n = 9$\\

Voor $n=5$:\\

$x_1 = -13 - 4\cdot(-2) = -5$\\

Voor $n=9$:\\

$x_1 = -13 - 8\cdot(-2) = 3$
