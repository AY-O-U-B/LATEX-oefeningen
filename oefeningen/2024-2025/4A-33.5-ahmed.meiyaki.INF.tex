$I = \displaystyle\int x^3 e^{x^2} dx $\\

$x^2 = t $\\

$\Rightarrow 2xdx = dt $\\

$\Rightarrow xdx = \dfrac{1}{2}dt $\\

$I = \dfrac{1}{2} \displaystyle\int t e^t dt $\\

$= \dfrac{1}{2} \left(e^t t - \displaystyle\int e^t dt\right) $\\

$= \dfrac{1}{2} e^t t - \dfrac{1}{2} e^t + C $\\

$= \dfrac{1}{2} e^t \left(t -  1\right) + C $\\

$= \dfrac{1}{2} e^{x^2} \left(x^2 - 1\right) + C $\\
