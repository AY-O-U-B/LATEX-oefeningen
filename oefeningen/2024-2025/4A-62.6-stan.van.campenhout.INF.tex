$ \displaystyle\int\dfrac{1}{x\sqrt{1-x^2}}dx \stackrel{x = \sin t}{=} \displaystyle\int\dfrac{1}{\sin t\sqrt{1-\sin^2t}}\cos tdt
= \displaystyle\int\dfrac{1}{\sin t\sqrt{\cos^2t}}\cos tdt $\\

$ = \displaystyle\int\dfrac{1}{\sin t}dt \stackrel{u = \tan\left(\tfrac{t}{2}\right)}{=} \displaystyle\int\dfrac{1}{\dfrac{u}{1+u^2}}\dfrac{2}{1+u^2}du 
= \displaystyle\int\dfrac{1}{u}du = \ln|u| + c $\\

$ \stackrel{u = \tan\left(\tfrac{t}{2}\right)}{=} \ln\left| \tan\left(\dfrac{t}{2}\right)\right| + c 
\stackrel{t = \operatorname{Bgsin}x}{=} \ln\left|\tan\left(\dfrac{\operatorname{Bgsin}x}{2}\right)\right| + c $\\
