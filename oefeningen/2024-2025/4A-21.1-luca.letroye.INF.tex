$\displaystyle\int \cos x \cosh x dx$\\

$\left\{
\begin{array}{l}
	u=\cos x \\ 
	dv=\cosh xdx
\end{array}
\right.
\Rightarrow 
\left\{
\begin{array}{l}
	du= 	-\sin xdx\\
 	v= \sinh x
\end{array}
\right.$\\

$\Rightarrow \cos x \sinh x - \displaystyle\int \sin x \sinh xdx$\\

$\left\{
\begin{array}{l}
	u=\sin x \\ 
	dv=\sinh xdx
\end{array}
\right.
\Rightarrow 
\left\{
\begin{array}{l}
	du= 	\cos xdx\\
 	v= \cosh x
\end{array}
\right.$\\

$\Rightarrow \cos x\sinh x + \sin x \cosh x - \displaystyle\int \cos x\cosh x dx$\\

$2I  = \cos x\sinh x + \sin x \cosh x +C $\\

$\Rightarrow I  = \dfrac{1}{2}(\cos x\sinh x + \sin x \cosh x )+C$\\
