$f(x,y) = x^2 +4x + y^3$\\

$\dfrac{\partial f}{\partial x} = 2x+4$\\

$\dfrac{\partial f}{\partial y} = 3y^2$\\

$\dfrac{\partial f}{\partial x} = 0 \Rightarrow 2x+4 = 0 \Rightarrow x = -2$\\

$\dfrac{\partial f}{\partial y} = 0 \Rightarrow 3y^2 = 0 \Rightarrow y = 0$\\

horizontaal raakvlak in het punt $(-2,0)$\\
