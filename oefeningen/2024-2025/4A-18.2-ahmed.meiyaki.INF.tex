$\displaystyle\int x^2 \sin x dx $\\

$= -x^2\cos x + \displaystyle\int 2x \cos x dx $\\

$= -x^2\cos x + 2x\sin x - \displaystyle\int 2\sin x dx $\\

$= -x^2\cos x + 2x\sin x - \displaystyle\int 2\sin x dx $\\

$= -x^2\cos x + 2x\sin x + 2\cos x + C $\\
