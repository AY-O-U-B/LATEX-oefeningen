$1 + \left(\log _{x-1} x\right)^{-1} = \left(\log _{2x+3} x\right)^{-1} + \log _x x \cdot \left(\log _2 x \right) $\\

$\Leftrightarrow 1 + \dfrac{1}{\log _{x-1} x}  = \dfrac{1}{\log _{2x+3} x} + \dfrac{\log _{2x+3} 2}{\log _{2x+3} x}$\\

$\Leftrightarrow 1 + \dfrac{1}{\log _{x-1} x} = \dfrac{1 + \log _{2x+3} 2 }{\log _{2x+3} x}$\\

$\Leftrightarrow 1 + \dfrac{\log _{2x+3} (x-1)}{\log _{2x+3} x} = \dfrac{1 + \log _{2x+3} 2 }{\log _{2x+3} x}$\\

$\Leftrightarrow 1 = \dfrac{1 - \log _{2x+3} (x-1) + \log _{2x+3} 2}{\log _{2x+3} x}$\\

$\Leftrightarrow 1 = \dfrac{\log _{2x+3} (2x+3) - \log _{2x+3} (x-1) + \log _{2x+3} 2}{\log _{2x+3} x} $\\

$\Leftrightarrow 1 = \dfrac{\log _{4x+6} (2x+3) - \log _{2x+3} (x-1)}{\log _{2x+3} x}$\\

$\Leftrightarrow \log _{2x+3} x = \log _{2x+3} \left(\dfrac{4x+6}{x-1}\right)$\\

$\Leftrightarrow x = \left(2x+3\right)^{\log _{2x+3} \left(\dfrac{4x+6}{x-1}\right)}$\\

$\Leftrightarrow x = \dfrac{4x+6}{x-1}$\\
