$ f_n : \mathbb{R} \rightarrow \mathbb{R}  : x \rightarrow \dfrac{\sin(nx)}{n} $\\

$ \lim\limits_{n \rightarrow \infty} \dfrac{\sin(nx)}{n} = 0 $\\

$ \Rightarrow f : \mathbb{R} \rightarrow \mathbb{R} : x \rightarrow 0 $ en $  f_n \stackrel{p}{\rightarrow} f $\\

$ \left|\left|f_n - f \right|\right|_{\infty} = \sup\limits_{x \in \mathbb{R}}\left|\dfrac{\sin(nx)}{n} - 0 \right| = \dfrac{1}{n} $\\

$ \Rightarrow \lim\limits_{n \rightarrow \infty}\left|\left|f_n - f \right|\right|_{\infty} = \lim\limits_{x \rightarrow \infty}\dfrac{1}{n} = 0 $\\

$ \Rightarrow f_n \stackrel{u}{\rightarrow} f $\\
