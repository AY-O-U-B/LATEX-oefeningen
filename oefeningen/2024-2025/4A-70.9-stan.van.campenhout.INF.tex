$ \displaystyle\int\dfrac{1}{x}\left(\ln\left(x\right)\right)^ndx = \displaystyle\int\left(\ln\left(x\right)\right)^nd\left(\ln\left(x\right)\right) $\\

$ \stackrel{n \not= -1}{=} \dfrac{\left(\ln\left(x\right)\right)^{n+1}}{n+1} + c $\\

$ \stackrel{n = -1}{=} \ln\left|\ln\left(x\right)\right| + c $\\

Als $ n \not= -1 \Rightarrow  \displaystyle\int\dfrac{1}{x}\left(\ln\left(x\right)\right)^ndx = \dfrac{\left(\ln\left(x\right)\right)^{n+1}}{n+1} + c $\\

Als $ n = -1 \Rightarrow \displaystyle\int\dfrac{1}{x}\left(\ln\left(x\right)\right)^ndx = \ln\left|\ln\left(x\right)\right| + c $\\
