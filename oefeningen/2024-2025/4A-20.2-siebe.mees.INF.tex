$ \displaystyle\int \sinh 5x dx $\\

$ = \displaystyle\int \dfrac{e^{5x}-e^{-5x}}{2} dx $\\

$ = \dfrac{1}{2}\displaystyle\int \left(e^{5x}-e^{-5x}\right) dx $\\

$ = \dfrac{1}{2}\left(\displaystyle\int e^{5x} dx - \displaystyle\int e^{-5x} dx\right) $\\

$ = \dfrac{1}{2}\left(\dfrac{1}{5}e^{5x} - \left(-\dfrac{1}{5}e^{-5x}\right)\right) + C $\\

$ = \dfrac{1}{2}\left(\dfrac{1}{5}e^{5x} + \dfrac{1}{5}e^{-5x}\right) + C $\\

$ = \dfrac{1}{10}\left(e^{5x}+e^{-5x}\right) + C $\\

$ = \dfrac{1}{5}\cosh 5x + C $\\
