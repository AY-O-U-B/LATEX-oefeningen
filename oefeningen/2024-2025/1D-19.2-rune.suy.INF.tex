$F(z) = 2z^3 - z^2 + 2z - 1$\\

\begin{tabular}{ c c c c c c }
                    &\vline     &$2$            &$-1$ &$2$  &$-1$ \\  
    $\dfrac{1}{2}$  &\vline     &$\downarrow$   &$1$  &$0$  &$1$ \\

\bigskip

$2z^2 + 2 = 0$\\

$\Leftrightarrow z^2 + 1 = 0$\\

$z^2 = -1$\\	

$z = \sqrt{-1}$\\

$z = \pm i$\\

$\Rightarrow F(z) = \left( z - \dfrac{1}{2}\right) \left(2z^2 + 2\right)$ is de ontbinding in factoren over $\mathbb{R}[z]$.\\

De nulpunten zijn $\dfrac{1}{2}, i, -i$\\
