$ z^4 - 7z^3 + 7z^2 - 7z + 7 \text{ voor } z = 3 + \sqrt{7} $\\

$ \Rightarrow (3 + \sqrt{7})^4 - 7(3 + \sqrt{7})^3 + 7(3 + \sqrt{7})^2 - 7(3 + \sqrt{7}) + 7 $\\

$ = (9 + 6\sqrt{7} + 7)^2 - 7(9 + 6\sqrt{7} + 7)(3 + \sqrt{7}) + 7(9 + 6\sqrt{7} + 7) - 21 -7\sqrt{7} + 7 $\\

$ = (256 + 192\sqrt{7} + 252) - 7(48 +18\sqrt{7} + 16\sqrt{7} + 42) + (112 + 42\sqrt{7}) - 21 - 7\sqrt{7} + 7$\\

$ = 508 + 192\sqrt{7} - 336 - 126\sqrt{7} - 112\sqrt{7} - 294 + 112 + 42\sqrt{7} - 14 - 7\sqrt{7} $\\

$ = -24 -11\sqrt{7} $\\ 

$\textbf{Resultaat: -24 - 11}\boldsymbol{\sqrt{7}}$
