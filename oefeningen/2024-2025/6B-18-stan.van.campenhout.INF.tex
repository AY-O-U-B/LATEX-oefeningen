$ s_n = n^3 - 2n^2 + 3n $\\

$ \Rightarrow $ hoogste graad is 3 dus zal de grootste som $ S_2 $ zijn \\

$ \Rightarrow s_n = aS_2(n) + bS_1(n) + n = \dfrac{1}{3}an^3+\dfrac{1}{2}an^2+\dfrac{1}{6}an+\dfrac{1}{2}bn^2 + \dfrac{1}{2}bn+cn $\\

$ = \dfrac{1}{3}an^3 + \left(\dfrac{1}{2}a+\dfrac{1}{2}b\right)n^2 + \left(\dfrac{1}{6}a + \dfrac{1}{2}b + c\right)n $\\

$ \Rightarrow \dfrac{1}{3}a = 1, \dfrac{1}{2}a + \dfrac{1}{2}b = -2, \dfrac{1}{6}a +\dfrac{1}{2}b + c = 3 $\\

$ \Rightarrow a = 3, a + b = 4, a + 3b + 6c = 18 $\\

$ \Rightarrow b = -7, 3 - 21 +6c = 18 \Rightarrow 6c = 36 \Rightarrow c = 6 $\\

$ \Rightarrow x_n = 3n^2 -7n + 6 $\\
