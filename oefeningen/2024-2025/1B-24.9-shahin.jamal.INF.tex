$1 + \dfrac{a - bi}{a + bi} + \dfrac{a + bi}{a - bi}$\\

\textbf{Eerste breuk:} $\dfrac{a - bi}{a + bi} \cdot \dfrac{a - bi}{a - bi} = \dfrac{(a - bi)^2}{(a + bi)(a - bi)}$\\

\textbf{Noemer:} $(a + bi)(a - bi) = a^2 - (bi)^2 = a^2 - (-b^2) = a^2 + b^2$\\

\textbf{Teller:} $(a - bi)^2 = a^2 - 2abi + (bi)^2 = a^2 - 2abi - b^2 = a^2 - b^2 - 2abi$\\

\textbf{Eerste breuk wordt: }$\dfrac{a^2 - b^2 - 2abi}{a^2 + b^2}$\\

\textbf{Tweede breuk: }$\dfrac{a + bi}{a - bi}$\\

$\dfrac{a + bi}{a - bi} \cdot \dfrac{a + bi}{a + bi} = \dfrac{(a + bi)^2}{(a - bi)(a + bi)}$\\

\textbf{Noemer: }$(a^2 + b^2)$\\

\textbf{Teller: }$(a + bi)^2 = a^2 + 2abi + (bi)^2 = a^2 + 2abi - b^2 = a^2 - b^2 + 2abi$\\

\textbf{Tweede breuk wordt: }$\dfrac{a^2 - b^2 + 2abi}{a^2 + b^2}$\\

\textbf{We krijgen dus:} \\

$1 + \dfrac{a^2 - b^2 - 2abi}{a^2 + b^2} + \dfrac{a^2 - b^2 + 2abi}{a^2 + b^2} = 1 + \dfrac{(a^2 - b^2 - 2abi) + (a^2 - b^2 + 2abi)}{a^2 + b^2}$\\

\textbf{Imaginaire delen vallen weg:} \\

$= 1 + \dfrac{2(a^2 - b^2)}{a^2 + b^2}$\\

$= \dfrac{a^2 + b^2}{a^2 + b^2} + \dfrac{2(a^2 - b^2)}{a^2 + b^2} = \dfrac{a^2 + b^2 + 2a^2 - 2b^2}{a^2 + b^2}$\\

\textbf{Uitkomst:} \\

$= \dfrac{3a^2 - b^2}{a^2 + b^2}$\\
