Oplossing: $a(z^2 + 1) + b(z + i) + c(iz^2 + 2z) = 3i$\\

$= az^2 + a + bz + bi + ciz^2 + 2cz$\\

$= (a + ci)z^2 + (b + 2c)z + (a + bi)$\\

Vergelijk met $3i$, dan volgt het stelsel:\\

$$
\left\{
\begin{array}{l}
a + ci = 0 \\
b + 2c = 0 \\
a + bi = 3i\\

\end{array}
\right.
$$

Uit de eerste vergelijking: $a = -ci$\\

Uit de tweede: $b = -2c$\\

Substitueer in de derde:\\

$a + bi = -ci + (-2c)i = -3ci = 3i \Rightarrow c = -1$\\

Daaruit volgt: $a = i$, $b = 2$, $c = -1$\\

\textbf{Antwoord:} $a = i$, $b = 2$, $c = -1$\\
