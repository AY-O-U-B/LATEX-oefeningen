$ \displaystyle\int x^3 \cos x dx = \displaystyle\int x^3 d\left(\sin x\right) = x^3\sin x - \displaystyle\int\sin x dx^3 $\\

$ =  x^3\sin x - 3\displaystyle\int x^2\sin x dx = x^3\sin x + 3\displaystyle\int x^2 d\left(\cos x\right) $\\

$ =  x^3\sin x + 3x^2\cos x - 6\displaystyle\int x d\left(\sin x\right) $\\

$ = x^3\sin x + 3x^2\cos x - 6x\sin x + 6\displaystyle\int\sin x dx $\\

$ = x^3\sin x + 3x^2\cos x - 6x\sin x -6\cos x + c $\\

$ = \left(x^3 - 6x\right)\sin x + \left(3x^2 - 6\right)\cos x + c $\\
