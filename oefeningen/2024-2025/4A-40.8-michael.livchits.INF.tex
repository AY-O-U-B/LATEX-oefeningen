$  \displaystyle\int\dfrac{dx}{\sqrt{2 - 3x^2}}$\\

$ = \dfrac{1}{\sqrt{3}}\displaystyle\int\dfrac{dx}{\sqrt{\dfrac{2}{3} - x^2}}$\\

$ = \dfrac{1}{\sqrt{3}}\displaystyle\int\dfrac{dx}{\sqrt{\sqrt{\dfrac{2}{3}}^2 - x^2}}$\\

$ = \dfrac{1}{\sqrt{3}}\operatorname{Bgsin}\dfrac{x}{\sqrt{\dfrac{2}{3}}} + c $\\

$ = \dfrac{\sqrt{3}}{3}\operatorname{Bgsin}\sqrt{\dfrac{3}{2}}\cdot x + c $\\
