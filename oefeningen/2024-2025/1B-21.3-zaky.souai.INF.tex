$\Leftrightarrow\dfrac{(2 + i)(3 + i)}{(3 - i)(3 + i)} = \dfrac{(1 + 3i)(4 - 2i)}{(4 + 2i)(4 - 2i)}$\\

$\Leftrightarrow\dfrac{6 + 2i + 3i + i^2}{9 - i^2} = \dfrac{4 - 2i + 12i - 6i^2}{16 - 4i^2}$\\

$\Leftrightarrow\dfrac{5 + 5i}{10} = \dfrac{10 + 10i}{20}$\\

$\Leftrightarrow\dfrac{1 + 1i}{2} = \dfrac{1 + 1i}{2}$\\
