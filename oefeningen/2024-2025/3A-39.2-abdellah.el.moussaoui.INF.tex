$f(x)=x^x$\\

$\Rightarrow y=x^x$\\

$\Rightarrow  \ln y=\ln x^x = x \ln x$\\

$\Rightarrow \left(\ln y\right)'= \left(x \ln x \right)'$\\

$\Rightarrow \dfrac{y'}{y} = \ln x + x \cdot \dfrac{1}{x}$\\

$\Rightarrow y' = y \left(\ln x + 1\right)$\\

$\Rightarrow \boxed{f'(x) = x^x \left(\ln x + 1\right)}$\\
