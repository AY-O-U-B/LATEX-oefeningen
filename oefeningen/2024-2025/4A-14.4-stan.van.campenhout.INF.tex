$ \displaystyle\int \ln^2x dx $\\

$ = x\ln^2x - \displaystyle\int x d\left(\ln^2x\right) = x\ln^2x - \displaystyle\int x \cdot 2\ln x \cdot \dfrac{1}{x} dx $\\

$ =  x\ln^2x - 2\displaystyle\int\ln x dx = x\ln^2x - 2x\ln x + 2\displaystyle\int x d\left(\ln x\right) $\\

$ =  x\ln^2x - 2x\ln x + 2x + c = x\left(\ln^2x - 2\ln x + 2\right) + c $\\
