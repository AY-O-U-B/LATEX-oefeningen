Opgave: $\dint \tan^2 x \left(1 + \tan^2 x\right) dx$\\

$= \dint \tan^2 x dx + \dint \tan^4 x dx$\\

$= \dint \tan^2 x dx + \dfrac{1}{3} \tan^3 x - \dint \tan^2 x dx$

$= \dfrac{1}{3} \tan^3 x + c$\\
