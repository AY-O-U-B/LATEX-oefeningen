Opgave: Bereken de Taylorpolynoom\\

$f(x, y) = e^{x^2 - y}$ in $(0, 0)$, graad $3$\\

$f(0, 0) = e^0 = 1$\\

Eerste orde parti\"ele afgeleiden:\\

$\dfrac{\partial f}{\partial x} = 2xe^{x^2 - y} \Rightarrow \dfrac{\partial f}{\partial x}(0, 0) = 0$\\

$\dfrac{\partial f}{\partial y} = -e^{x^2 - y} \Rightarrow \dfrac{\partial f}{\partial y}(0, 0) = -1$\\

Tweede orde parti\"ele afgeleiden:\\

$\dfrac{\partial^2 f}{\partial x^2} = 2e^{x^2 - y} + 4x^2 e^{x^2 - y} \Rightarrow \dfrac{\partial^2 f}{\partial x^2}(0, 0) = 2$\\

$\dfrac{\partial^2 f}{\partial y^2} = e^{x^2 - y} \Rightarrow \dfrac{\partial^2 f}{\partial y^2}(0, 0) = 1$\\

$\dfrac{\partial^2 f}{\partial x\partial y} = \dfrac{\partial^2 f}{\partial y\partial x} = -2xe^{x^2 - y} \Rightarrow \dfrac{\partial^2 f}{\partial x\partial y}(0, 0) = \dfrac{\partial^2 f}{\partial y\partial x}(0, 0) = 0$\\

Derde orde parti\"ele afgeleiden:\\

$\dfrac{\partial^3 f}{\partial x^3} = 4xe^{x^2 - y} + 8xe^{x^2 - y} + 8x^3 e^{x^2 - y} = \left(12x + 8x^3\right)e^{x^2 - y} \Rightarrow \dfrac{\partial^3 f}{\partial x^3}(0, 0) = 0$\\

$\dfrac{\partial^3 f}{\partial y^3} = -e^{x^2 - y} \Rightarrow \dfrac{\partial^3 f}{\partial y^3}(0, 0) = -1$\\

$\dfrac{\partial^3 f}{\partial x^2 \partial y} = \dfrac{\partial^3 f}{\partial y \partial x^2} = \dfrac{\partial^3 f}{\partial x \partial y \partial x} = -2e^{x^2 - y} -4x^2e^{x^2 - y}$\\

$\Rightarrow \dfrac{\partial^3 f}{\partial x^2 \partial y}(0, 0) = \dfrac{\partial^3 f}{\partial y \partial x^2}(0, 0) = \dfrac{\partial^3 f}{\partial x \partial y \partial x}(0, 0) = -2$\\

$\dfrac{\partial^3 f}{\partial y^2 \partial x} = \dfrac{\partial^3 f}{\partial x \partial y^2} = \dfrac{\partial^3 f}{\partial y \partial x \partial y} = 2xe^{x^2 - y}$\\

$\Rightarrow \dfrac{\partial^3 f}{\partial y^2 \partial x}(0, 0) = \dfrac{\partial^3 f}{\partial x \partial y^2}(0, 0) = \dfrac{\partial^3 f}{\partial y \partial x \partial y}(0, 0) = 0$\\
