$\lim\limits_{x \rightarrow \infty} \dfrac{\sqrt{x^3 + 4}}{x}$\\

$= \lim\limits_{x \rightarrow \infty} \dfrac{\sqrt{x^3 \left(1 + \dfrac{4}{x^3}\right)}}{x}$\\

$= \lim\limits_{x \rightarrow \infty} \dfrac{\left |x \right|\sqrt{x \left(1 + \dfrac{4}{x^3}\right)}}{x}$\\

\medskip
$\lim\limits_{x \rightarrow +\infty} \dfrac{\left |x \right|\sqrt{x \left(1 + \dfrac{4}{x^3}\right)}}{x}$\\

$ = \lim\limits_{x \rightarrow +\infty} \dfrac{x \sqrt{x \left(1 + \dfrac{4}{x^3}\right)}}{x}$\\

$ = \lim\limits_{x \rightarrow +\infty}  \sqrt{x \left(1 + \dfrac{4}{x^3}\right)}= +\infty$\\
\medskip

$\lim\limits_{x \rightarrow -\infty} \dfrac{\left |x \right|\sqrt{x \left(1 + \dfrac{4}{x^3}\right)}}{x}$\\

$ = \lim\limits_{x \rightarrow -\infty} \dfrac{-x \sqrt{x \left(1 + \dfrac{4}{x^3}\right)}}{x}$\\

$ = \lim\limits_{x \rightarrow -\infty}  -\sqrt{x \left(1 + \dfrac{4}{x^3}\right)}$
bestaat niet!
