$= \dfrac{1}{4}\displaystyle\int (1 + 2\cos2x + \cos^22x)dx = \dfrac{1}{4}x + \dfrac{1}{4}\sin 2x + \dfrac{1}{4}\displaystyle\int \left(\dfrac{1+\cos 4x}{2}\right) dx$\\

$= \dfrac{1}{4}x + \dfrac{1}{4}\sin 2x + \dfrac{1}{8} \displaystyle\int (1 + \cos4x) dx = \dfrac{1}{4}x + \dfrac{1}{4}\sin 2x + \dfrac{1}{8}x + \dfrac{1}{32}\sin4x$\\

$= \dfrac{3}{8}x + \dfrac{1}{4}\sin 2x + \dfrac{1}{32}\sin 4x + c$\\
