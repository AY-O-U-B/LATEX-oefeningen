$ \displaystyle\int\dfrac{dx}{\sqrt{3 + 2x - x^2}}$\\

$ = \displaystyle\int\dfrac{dx}{\sqrt{-(x^2 - 2x + 1 - 4)}}$\\

$ = \displaystyle\int\dfrac{dx}{\sqrt{-(x^2 - 2x + 1 - 4)}}$\\

$ = \displaystyle\int\dfrac{dx}{\sqrt{-((x - 1)^2 - 4)}}$\\

$ = \displaystyle\int\dfrac{dx}{\sqrt{4 - (x-1)^2}}$\\

$ = \displaystyle\int\dfrac{du}{\sqrt{4 - u^2}}$\\

$ = \displaystyle\int\dfrac{du}{\sqrt{2^2 - u^2}}$\\

$ = \operatorname{Bgsin}\dfrac{u}{2} + c $\\

$ = \operatorname{Bgsin}\dfrac{x-1}{2} + c $\\
