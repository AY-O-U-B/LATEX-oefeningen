$ \lim\limits_{x \rightarrow \tfrac{\pi}{2}} \left(\dfrac{\cos x}{1 - \sin x} \right)
= \lim\limits_{x \rightarrow \tfrac{\pi}{2}} \left(\dfrac{\cos x ( 1 + \sin x)}{(1 - \sin x)(1 + \sin x)} \right) 
= \lim\limits_{x \rightarrow \tfrac{\pi}{2}} \left(\dfrac{\cos x + \sin x \cos x}{1 - \sin^2 x} \right) $\\

\medskip


$ = \lim\limits_{x \rightarrow \tfrac{\pi}{2}} \left(\dfrac{\cos x + \sin x \cos x}{\cos^2 x} \right)  
=  \lim\limits_{x \rightarrow \tfrac{\pi}{2}} \left(\dfrac{1 + \sin x}{\cos x} \right) = \dfrac{2}{0} = \infty $\\

Bekijk linker en rechter limiet \\

$ \lim\limits_{x \rightarrow \tfrac{\pi}{2}-} \left(\dfrac{1 + \sin x}{\cos x} \right) = +\infty $\\

$ \lim\limits_{x \rightarrow \tfrac{\pi}{2}+} \left(\dfrac{1 + \sin x}{\cos x} \right) = -\infty $\\
