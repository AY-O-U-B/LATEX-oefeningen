$\displaystyle\int\dfrac{\sin x + x\cos x}{x\sin x} \ dx$\\

$= \displaystyle\int\dfrac{\sin x}{x\sin x} \ dx + \displaystyle\int\dfrac{x\cos x}{x\sin x} \ dx = \displaystyle\int\dfrac{1}{x} \ dx + \displaystyle\int\dfrac{\cos x}{\sin x} \ dx$\\

$= \ln \lvert x \rvert + c + \displaystyle\int\dfrac{\cos x}{\sin x} \ dx$\\

$u = \sin x \Rightarrow du = \cos x \ dx$\\

$\Rightarrow \ln \lvert x \rvert + c + \displaystyle\int\dfrac{1}{u} \ du$\\

$=\ln \lvert x \rvert + \ln \lvert \sin x \rvert + c$\\
