\newcommand{\llim}{\lim\limits}
\newcommand{\dint}{\displaystyle\int}

Opgave: $\left(f_n = n \cdot 1_{\left]0, \tfrac{1}{n}\right]}\right)_{n\in\mathbb{N}_0}$\\

Puntsgewijze convergentie naar nulfunctie:\\

Stel $x \not\in \left]0, \dfrac{1}{n}\right] \Rightarrow f_n(x)  = 0$\\

$\forall x > 0, \exists N\in\mathbb{N}_0: f_n(x) = 0, \forall n \ge N \Rightarrow \llim_{n\to\infty} f_n(x) = \llim_{n\to\infty} \left (n \cdot 1_{\left]0, \tfrac{1}{n}\right]}(x)\right) = 0$\\

Geen convergentie naar nulfunctie in kwadratisch gemiddelde:\\

$\llim_{n\to\infty} \dint_{0+}^{\tfrac{1}{n}} |f_n(x) - f(x)|^2 dx = \llim_{n\to\infty} \dint_{0+}^{\tfrac{1}{n}} |f_n(x) - 0|^2 dx = \llim_{n\to\infty} \left(n^2 \dint_{0+}^{\tfrac{1}{n}} \left|\left(1_{\left]0, \tfrac{1}{n}\right]}(x)\right)\right|^2 dx\right)$\\

$= \llim_{n\to\infty} \left(n^2\llim_{a\to 0+}\dint_{a}^{\tfrac{1}{n}} dx\right) = \llim_{n\to\infty} \left(n^2[x]_{0}^{\tfrac{1}{n}}\right) = \llim_{n\to\infty} \dfrac{n^n}{n}$\\

$= \llim_{n\to\infty} n = \infty \Rightarrow$ divergent\\
