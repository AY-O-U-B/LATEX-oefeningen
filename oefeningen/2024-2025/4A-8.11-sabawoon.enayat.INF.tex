Opgave: $\dint \dfrac{dx}{\tan (3x + 5)}$\\

$\Rightarrow t = 3x + 5$, $t' = 3$, $dt = t'dx \Rightarrow dx = \dfrac{dt}{3}$\\

$= \dint \dfrac{dt}{3\tan t} = \dfrac{1}{3}\dint \dfrac{1}{\dfrac{\sin t}{\cos t}} dt = \dfrac{1}{3}\dint \dfrac{\cos t}{\sin t} dt$\\

$\Rightarrow u = \sin (t)$, $u' = \cos (t)$, $du = u'dt = \cos (t) dt \Rightarrow dt = \dfrac{dt}{u'}$\\

$\Rightarrow \dfrac{\cos t}{\sin t} \dfrac{1}{\cos t} = \dfrac{1}{\sin t} = \dfrac{1}{u}$\\

$= \dfrac{1}{3} \dint \dfrac{1}{u} du = \dfrac{1}{3} \ln\left(|u|\right) + c$\\

$= \dfrac{1}{3} \ln\left(|\sin t|\right) + c = \dfrac{1}{3} \ln\left(|\sin (3x + 5)|\right) + c$\\
