$ \displaystyle\int\dfrac{\sqrt{x}-1}{\left(\sqrt[3]{x}-1\right)\sqrt[6]{x}}dx \stackrel{t = \sqrt[6]{x}}{=} \displaystyle\int\dfrac{t^3-1}{t\left(t^2-1\right)}\cdot6t^5dt $\\

$ = 6\displaystyle\int\dfrac{t^8-t^5}{t\left(t^2-1\right)}dt = 6\displaystyle\int\left(t^5+t^3-t^2+t-1 + \dfrac{t^2-t}{t^3-t}\right)dt $\\

$ = t^6+\dfrac{3}{2}t^4-2t^3+3t^2-6t + 6\displaystyle\int\dfrac{t\left(t-1\right)}{t\left(t-1\right)\left(t+1\right)}dt $\\

$ = t^6+\dfrac{3}{2}t^4-2t^3+3t^2-6t + 6\ln|t+1| + c $\\ 

$ \stackrel{t = \sqrt[6]{x}}{=} x+\dfrac{3}{2}x^{\tfrac{2}{3}} -2x^{\tfrac{1}{2}} +3x^{\tfrac{1}{3}} -6x^{\tfrac{1}{6}} + 6\ln|x^{\tfrac{1}{6}} +1| + c $\\
