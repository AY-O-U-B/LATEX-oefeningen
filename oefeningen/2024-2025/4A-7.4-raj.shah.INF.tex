$\displaystyle\int \dfrac{\sqrt[5]{x^3} + \sqrt[6]{x^5}}{\sqrt[4]x} dx$\\

$ = \displaystyle\int \dfrac{x^{\tfrac{3}{5}}}{x^{\tfrac{1}{4}}}dx + \displaystyle\int \dfrac{x^{\tfrac{5}{6}}}{x^{\tfrac{1}{4}}}dx = \displaystyle\int \dfrac{x^{\tfrac{12}{20}}}{x^{\tfrac{5}{20}}}dx + \displaystyle\int \dfrac{x^{\tfrac{20}{24}}}{x^{\tfrac{6}{24}}}dx $\\

$ = \displaystyle\int x^{\tfrac{7}{20}}dx + \displaystyle\int x^{\tfrac{14}{24}}dx$\\

$ = \dfrac{20x^{\tfrac{27}{20}}}{27} + \dfrac{24x^{\tfrac{38}{24}}}{38} + c = \dfrac{20x \sqrt[20]{x^7}}{27} + \dfrac{12x\sqrt[12]{x^7}}{19} + c$
