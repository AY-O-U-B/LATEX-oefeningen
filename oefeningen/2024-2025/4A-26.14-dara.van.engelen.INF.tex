$\displaystyle\int (x^2+1)(x+1)^5dx$\\

Stel $ x+1 = t \Rightarrow dx = dt$\\

$ = \displaystyle\int ((t-1)^2 + 1)t^5dt = \displaystyle\int (t^2 - 2t +2)t^5dt = \displaystyle\int (t^7-2t^6 +2t^5)dt$\\

$ = \dfrac{t^8}{8} - \dfrac{2t^7}{7} + \dfrac{2t^6}{6} + C = \dfrac{(x+1)^8}{8} - \dfrac{2(x+1)^7}{7} + \dfrac{2(x+1)^6}{6} + C$\\

$ = \dfrac{(x+1)^8}{8} - \dfrac{2(x+1)^7}{7} + \dfrac{(x+1)^6}{3} + C$\\
