Bereken de afgeleide van: $f(x) = e^x (\sin x - \cos x)$\\

$f'(x) = e^x \cdot d(\sin x - \cos x) + e^x (\sin x - \cos x)$\\

$= e^x (\cos x + \sin x) + e^x (\sin x - \cos x)$\\

$= e^x (\cos x + \sin x + \sin x - \cos x) = 2 e^x \sin x$\\
