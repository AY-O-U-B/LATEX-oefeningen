$ 1 + \dfrac{a - bi}{a + bi} + \dfrac{a + bi}{a - bi} = 1 + \dfrac{(a - bi)^2}{a^2 - (bi)^2} + \dfrac{(a + bi)^2}{a^2 - (bi)^2} $\\

$ = 1 + \dfrac{(a - bi)^2}{a^2 + b^2} + \dfrac{(a + bi)^2}{a^2 + b^2} $\\

$ = 1 + \dfrac{a^2 - b^2 - 2abi}{a^2 + b^2} + \dfrac{a^2 - b^2 + 2abi}{a^2 + b^2} $\\

$ = \dfrac{a^2 + b^2 + a^2 - b^2 - 2abi + a^2 - b^2 + 2abi}{a^2 + b^2} $\\

$ = \dfrac{3a^2 - b^2}{a^2 + b^2} $
