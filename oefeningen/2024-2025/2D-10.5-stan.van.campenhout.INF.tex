$ \lim\limits_{x \rightarrow -\tfrac{1}{3}} \left(\dfrac{6x^2 + 11x + 3}{6x^2 - 7x - 3} \right) 
= \lim\limits_{x \rightarrow -\tfrac{1}{3}} \left(\dfrac{(3x + 1)(2x + 3)}{(3x + 1)(2x - 3)} \right) 
= \lim\limits_{x \rightarrow -\tfrac{1}{3}} \left(\dfrac{2x + 3}{2x - 3} \right)  = \dfrac{-\dfrac{2}{3} + 3}{-\dfrac{2}{3} - 3}
= \dfrac{\dfrac{7}{3}}{\dfrac{-11}{3}} = -\dfrac{7}{11} $\\
