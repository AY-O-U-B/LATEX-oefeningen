$\displaystyle\int \dfrac{x^2 -2x +4}{x^2 -2x +1}dx$\\

$ = \displaystyle\int \dfrac{x^2 -2x +1}{x^2 -2x +1}dx + \displaystyle\int \dfrac{3}{x^2 -2x +1}dx$\\

$ = \displaystyle\int dx + 3\displaystyle\int \dfrac{1}{x^2 -2x +1}dx$\\

$ = \left(x + 3\cdot \displaystyle\int \dfrac{1}{x^2 -2x +1}\right) dx$\\

$ = \left(x + 3\cdot \displaystyle\int \dfrac{1}{(x-1)^2}\right) dx$\\

Stel $t = x-1$ dan $dx = dt$\\

$ = \left(x+3\displaystyle\int \dfrac{1}{t^2}\right)dt = \left(x+3\displaystyle\int t^{-2} \right)dt$\\

$ = x + 3 \left(\dfrac{-1}{t}\right) +c $\\

$ = x - \dfrac{3}{x-1} + c$\\
