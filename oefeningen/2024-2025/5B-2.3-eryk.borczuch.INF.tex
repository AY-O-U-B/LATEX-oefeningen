$\displaystyle\int_{-\infty}^{+\infty} e^{-x} dx $\\

$= \lim\limits_{a \to -\infty } \displaystyle\int_{a}^{0}e^{-x}dx + \lim\limits_{a \to +\infty } \displaystyle\int_{0}^{b} e^{-x}dx $\\

$= \lim\limits_{a \to -\infty} [-e^{-x}]_a^0 + \lim\limits_{a \to +\infty} [-e^{-x}]_0^b $\\

$= \lim\limits_{a \to -\infty} (-e^0 + e^{-a} ) + \lim\limits_{a \to +\infty} (-e^{-b} + e^{-0})  $\\

$= -1 + e^{+\infty} + 0 + 1 $\\

$= +\infty$\\
