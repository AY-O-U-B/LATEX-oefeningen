	Bereken de afgeleide functie van
	
	$f(x) = \dfrac{2x^2 - 4x + 1}{\sqrt{x}}$\\
	
	$f'(x) = \left( 2x^2 - 4x + 1 \right)' x^{-\tfrac{1}{2}} + \left( 2x^2 - 4x + 1 \right) \left(x^{-\tfrac{1}{2}}\right)'$\\
	
	 $= (4x - 4) x^{-\tfrac{1}{2}} + (2x^2 - 4x + 1) \left(-\dfrac{1}{2} x^{-\tfrac{3}{2}} \right)$\\
			
	$ f'(x) = \dfrac{4x-4}{\sqrt{x}} - \dfrac{2x^2 - 4x + 1}{2x^{\tfrac{3}{2}}} $\\ 
	
	$ = \dfrac{4(x-1)}{x^{\tfrac{1}{2}}} - \dfrac{1}{2} \left( \dfrac{2x^2}{x^{\tfrac{3}{2}}} - \dfrac{4x}{x^{\tfrac{3}{2}}} + \dfrac{1}{x^{\tfrac{3}{2}}} \right) $\\
	
	$ = \left( 4x^{\tfrac{1}{2}} - 4x^{-\tfrac{1}{2}} \right) - \dfrac{1}{2} \left( 2x^{\tfrac{1}{2}} - 4x^{-\tfrac{1}{2}} + x^{-\tfrac{3}{2}} \right) $\\
	
	$ = 4x^{\tfrac{1}{2}} - 4x^{-\tfrac{1}{2}} - x^{\tfrac{1}{2}} + 2x^{-\tfrac{1}{2}} - \dfrac{1}{2} x^{-\tfrac{3}{2}} $\\
	
	$ = (4x^{\tfrac{1}{2}} - x^{\tfrac{1}{2}}) + (-4x^{-\tfrac{1}{2}} + 2x^{-\tfrac{1}{2}}) - \dfrac{1}{2} x^{-\tfrac{3}{2}} $\\
	
	$ = 3x^{\tfrac{1}{2}} - 2x^{-\tfrac{1}{2}} - \dfrac{1}{2} x^{-\tfrac{3}{2}} $\\  
	
	$ = 3\sqrt{x} - \dfrac{2}{\sqrt{x}} - \dfrac{1}{2x^{\tfrac{3}{2}}} $
