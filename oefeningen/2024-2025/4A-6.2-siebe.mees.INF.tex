$\displaystyle\int \dfrac{-x^2+x+1}{x^4} dx$\\

$ = \displaystyle\int \left( \dfrac{-x^2}{x^4} + \dfrac{x}{x^4} + \dfrac{1}{x^4} \right) dx$\\

$ = \displaystyle\int \left( -\dfrac{1}{x^2} + \dfrac{1}{x^3} + \dfrac{1}{x^4} \right) dx$\\

$ = \displaystyle\int \left( -x^{-2}+x^{-3}+x^{-4} \right) dx $\\

$ = \displaystyle\int -x^{-2}dx + \displaystyle\int x^{-3}dx + \displaystyle\int x^{-4} dx $\\

$ = \dfrac{1}{x} - \dfrac{1}{2x^2} - \dfrac{1}{3x^3} + C $\\
