opgave: $\displaystyle\int \dfrac{3x^2+x+4}{x^3+x^2+x+1}dx$\\

$\Rightarrow \dfrac{3x^2+x+4}{(x+1)(x^2+1)} = \dfrac{A}{x+1} + \dfrac{Bx + C}{x^2 + 1}$\\

$\Rightarrow A = \left .\dfrac{3x^2+x+4}{x^2 + 1} \right|_{x = -1} = 3$\\

$\Rightarrow Bi + C = \left .\dfrac{3x^2+x+4}{x+1} \right|_{x = i} = \dfrac{i+1}{i+1} = 1$\\

$\dfrac{3x^2+x+4}{(x+1)(x^2+1)} = \dfrac{3}{x+1} + \dfrac{1}{x^2 + 1}$\\

$I = \displaystyle\int \dfrac{3}{x+1}dx + \displaystyle\int \dfrac{1}{x^2 + 1}dx$\\

$= 3 \ln|x+1| + \operatorname{Bgtan}(x) + c$\\
