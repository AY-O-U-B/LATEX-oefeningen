$ \displaystyle\int\dfrac{8\left(x^3-1\right)}{9x^4+10x^2+1}dx = 8\displaystyle\int\dfrac{x^3-1}{9x^4+10x^2+1}dx 
= \dfrac{8}{9}\displaystyle\int\dfrac{x^3-1}{\left(x^2+1\right)\left(x^2+\dfrac{1}{9}\right)}dx $\\

$ \rightarrow \dfrac{x^3-1}{\left(x^2+1\right)\left(x^2+\dfrac{1}{9}\right)} = \dfrac{Ax+B}{x^2+1}+\dfrac{Cx+D}{x^2+\dfrac{1}{9}} $\\

$ \Rightarrow A\left(i\right)+B = \dfrac{i^3-1}{i^2+\dfrac{1}{9}} = \dfrac{9}{8}+\dfrac{9}{8}i \Rightarrow A = \dfrac{9}{8}, B = \dfrac{9}{8} $\\

$ \Rightarrow C\left(\dfrac{1}{3}i\right)+D = \dfrac{-\dfrac{1}{27}i-1}{-\dfrac{1}{9}+1} = -\dfrac{9}{8}-\dfrac{1}{24}i \Rightarrow C = -\dfrac{1}{8}, D = -\dfrac{9}{8} $\\

$ = \dfrac{8}{9}\displaystyle\int\dfrac{\dfrac{9}{8}x+\dfrac{9}{8}}{x^2+1}dx+\dfrac{8}{9}\displaystyle\int\dfrac{-\dfrac{1}{8}x-\dfrac{9}{8}}{x^2+\dfrac{1}{9}}dx $\\

$ = \dfrac{1}{9}\displaystyle\int\dfrac{\dfrac{9}{2}\left(2x\right)+9}{x^2+1}dx + \dfrac{1}{9}\displaystyle\int\dfrac{-\dfrac{1}{2}\left(2x\right)-9}{x^2+\dfrac{1}{9}}dx $\\

$ = \dfrac{1}{2}\ln|x^2+1|+\operatorname{Bgtan}x-\dfrac{1}{18}\ln\left|x^2+\dfrac{1}{9}\right|-\dfrac{1}{\dfrac{1}{3}}\operatorname{Bgtan}\left(\dfrac{1}{\dfrac{1}{3}}x\right)+c $\\

$ = \dfrac{1}{2}\ln|x^2+1|+\operatorname{Bgtan}x-\dfrac{1}{18}\ln\left|x^2+\dfrac{1}{9}\right|-3\operatorname{Bgtan}\left(3x\right)+c $\\
