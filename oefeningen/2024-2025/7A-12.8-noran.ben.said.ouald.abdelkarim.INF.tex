$f(x, y, z) = xy + yz + zx$ in a $\left(1,-1,2\right)$ in de richting $\left(1,1,1\right)$\\

$\dfrac{\partial f}{\partial x}\left(x,y,z\right) = y + z \Rightarrow \dfrac{\partial f}{\partial x}\left(1,-1,2\right) = -1 + 2 = 1$\\

$\dfrac{\partial f}{\partial y}\left(x,y,z\right) = x + z \Rightarrow \dfrac{\partial f}{\partial y}\left(1,-1,2\right) = 1+2 = 3$\\

$\dfrac{\partial f}{\partial z}\left(x,y,z\right) = y + x \Rightarrow \dfrac{\partial f}{\partial z}\left(1,-1,2\right) = -1 + 1=0$\\

$\Rightarrow D f(\bar a, \bar h) = D f(\bar a)\bar h = \left( 
\begin{array}{ccc}
1 & 3 & 0
\end{array}
\right)
\left( 
\begin{array}{ccc}
1\\
1\\
1
\end{array}
\right)
= 4$\\
