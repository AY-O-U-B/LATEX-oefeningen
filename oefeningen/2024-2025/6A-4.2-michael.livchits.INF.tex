$x_4 + x_6 = 30$ en $x_9 + x_{11} = 50$\\

$x_4 = x_1 + 3v$ en $x_6 = x_1 + 5v$\\

$\Rightarrow 30 = x_1 + 3v + x_1 + 5v = 2x_1 + 8v$\\

$x_9 = x_1 + 8v$ en $x_{11} = x_1 + 10v$\\

$\Rightarrow 50 = x_1 + 8v + x_1 + 10v = 2x_1 + 18v$\\

$\left\{
	\begin{array}{l}
		2x_1 + 8v = 30 \\
		2x_1 + 18v = 50
	\end{array}
\right .$\\

$10v = 20$\\

$v = 2$\\

$v$ invullen geeft dan:\\

$2x_1 + 16 = 30$\\

$x_1 = 7$ en $v = 2$\\
