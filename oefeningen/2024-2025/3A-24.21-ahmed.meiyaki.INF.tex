$f(x) = \dfrac{\sin x - \cos x}{\sin x + \cos x} $\\

$f'(x) = \dfrac{\left(\cos x + \sin x\right)\left(\sin x + \cos x\right) - \left(\sin x - \cos x\right)\left(\cos x - \sin x\right)}{\left(\sin x + \cos x\right)^2} $\\

$= \dfrac{\left(\cos x + \sin x\right)^2 + \left(\sin x - \cos x\right)^2}{\left(\sin x + \cos x\right)^2} $\\

$= \dfrac{\cos^2x + 2\cos x \sin x + \sin^2x + \sin^2x - 2\sin x \cos x + \cos^2x}{\sin^2x + 2\cos x \sin x + \cos^2x} $\\

$= \dfrac{2}{1 + \sin\left(2x\right)} $\\
