$ \displaystyle\int\sin x\cos x dx $\\

17.1) \\

$ = \displaystyle\int\sin xd\left(\sin x\right) \stackrel{t = \sin x}{=} \displaystyle\int tdt = \dfrac{1}{2}t^2 \stackrel {t = \sin x} = \dfrac{1}{2}\sin^2x + c $\\

17.2) \\

$ = -\displaystyle\int\cos xd\left(\cos x\right) \stackrel{t = \cos x}{=} -\displaystyle\int tdt = -\dfrac{1}{2}t^2 \stackrel{t = \cos x}{=} -\dfrac{1}{2}\cos^2x + c $\\

17.3) \\

$ = \dfrac{1}{2}\displaystyle\int\sin\left(2x\right)dx = \dfrac{1}{4}\displaystyle\int\sin\left(2x\right)d\left(2x\right) = -\dfrac{1}{4}\cos\left(2x\right) + c $\\
