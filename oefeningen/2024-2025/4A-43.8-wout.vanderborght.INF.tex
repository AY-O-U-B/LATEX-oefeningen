$ \Int \dfrac{2x^2+3x+3}{(x+1)^3}dx $ \\

$ \dfrac{2x^2+3x+3}{(x+1)^3}dx = \dfrac{A}{(x+1)^3} + \dfrac{B}{(x+1)^2} + \dfrac{C}{x+1} $ \\

$ A = \left. 2x^2+3x+3 \right|_{x = -1} = 2 $ \\

$ \dfrac{2x^2+3x+3}{(x+1)^3} - \dfrac{2}{(x+1)^3} = \dfrac{2x^2+3x+2}{(x+1)^3} = \dfrac{(x+1)(2x+1)}{(x+1)^3} = \dfrac{2x+1}{(x+1)^2} $ \\

$ B = \left. 2x+1 \right|_{x = -1} = -1 $ \\

$ \dfrac{2x+1}{(x+1)^2} - \dfrac{-1}{(x+1)^2} = \dfrac{2(x+1)}{(x+1)^2} = \dfrac{2}{x+1} $ \\

$ C = \left. 2 \right|_{x = -1} = 2 $ \\

$ \Int \dfrac{2x^2+3x+3}{(x+1)^3}dx = \Int \dfrac{2dx}{(x+1)^3} - \Int \dfrac{dx}{(x+1)^2} + \Int \dfrac{2dx}{x+1} $ \\

Stel $t = x+1 \Rightarrow dt = dx $\\

$ = 2\Int \dfrac{dt}{t^3} - \Int \dfrac{dt}{t^2} + 2\Int \dfrac{dt}{t} $ \\

$ = 2\dfrac{t^{-2}}{-2} - \dfrac{t^{-1}}{-1} + 2\ln|t| + C $

$ = \dfrac{-1}{t^2} + \dfrac{1}{t} + 2\ln|t| + C $ \\

$ = \dfrac{-1}{(x+1)^2} + \dfrac{1}{x+1} + 2\ln|x+1| + C $ \\
