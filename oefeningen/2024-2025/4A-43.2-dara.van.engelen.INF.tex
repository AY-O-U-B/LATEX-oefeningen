$ I = \displaystyle\int \dfrac{x^2+7x+1}{(x+2)^2(2x+1)}dx$\\\\

$\dfrac{x^2+7x+1}{(x+2)^2(2x+1)} = \dfrac{A}{2x+1} + \dfrac{B}{(x+2)^2} + \dfrac{C}{x+2}$\\\\

$A  =\left. \dfrac{x^2+7x+1}{(x+2)^2} \right|_{x=\tfrac{-1}{2}} = -1$\\\\

$B  =\left. \dfrac{x^2+7x+1}{2x+1} \right|_{x=-2}= 3$\\\\

$ \left( \dfrac{x^2+7x+1}{(x+2)^2(2x+1)} - \dfrac{3}{(x+2)^2}\right)(x+2)$\\\\

$ = \left( \dfrac{(x^2+7x+1)-3(2x+1)}{(x+2)^2(2x+1)} \right)(x+2) = \dfrac{x^2+x-2}{(x+2)(2x+1)} $\\\\

$= \dfrac{(x+2)(x-1)}{(x+2)(2x+1)} = \dfrac{x-1}{2x+1}$\\\\

$\Rightarrow C = \left. \dfrac{x-1}{2x+1} \right|_{x=-2} = 1$\\\\

$\Rightarrow I = \displaystyle\int \dfrac{-dx}{2x+1} + \displaystyle\int \dfrac{3dx}{(x+2)^2} + \displaystyle\int \dfrac{dx}{x+2}$\\\\

$ = -\displaystyle\int \dfrac{dx}{2x+1} + 3\displaystyle\int \dfrac{dx}{(x+2)^2} + \displaystyle\int \dfrac{dx}{x+2}$\\\\

$ = -\dfrac{\ln|2x+1|}{2} - \dfrac{3}{x+2} + \ln|x+2| + C$
