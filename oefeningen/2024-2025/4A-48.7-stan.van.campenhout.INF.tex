$ \displaystyle\int\dfrac{6}{e^{2x}+e^x-2}dx \stackrel{t = e^x}{=} 6\displaystyle\int\dfrac{1}{t^2+t+2}\cdot\dfrac{1}{t}dt 
= 6\displaystyle\int\dfrac{1}{t\left(t-1\right)\left(t+2\right)}dt  $\\

$ \Rightarrow \dfrac{1}{t\left(t-1\right)\left(t+2\right)} = \dfrac{A}{t} + \dfrac{B}{t-1} + \dfrac{C}{t+2} $\\

$ \Rightarrow A = \dfrac{1}{-1\cdot2} = -\dfrac{1}{2} $\\

$ \Rightarrow B = \dfrac{1}{1\left(1+2\right)} = \dfrac{1}{3} $\\

$ \Rightarrow C = \dfrac{1}{-2\left(-2-1\right)} = \dfrac{1}{6} $\\

$ = -3\displaystyle\int\dfrac{1}{t}dt + 2\displaystyle\int\dfrac{1}{t-1}dt + \displaystyle\int\dfrac{1}{t+2}dt $\\

$ = -3\ln|t| + 2\ln|t-1| + \ln|t+2| + c $\\

$ \stackrel{t = e^x}{=} -3x + 2\ln|e^x-1| + \ln|e^x+2| + c $\\
