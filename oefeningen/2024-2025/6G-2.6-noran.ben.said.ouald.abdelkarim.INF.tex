$ f(x) =
\left\{
\begin{array}{ll}
0 & \text{als } x \in [-\pi,0] \\
2\sin x & \text{als } x \in ]0,\pi]
\end{array}
\right.$\\

$ a_0 = \dfrac{1}{\pi}\displaystyle\int_{-\pi}^{\pi} f(x)\,dx$\\

$ = \dfrac{1}{\pi}\displaystyle\int_0^{\pi} 2\sin x\,dx$\\

$ = \dfrac{2}{\pi}\left[-\cos x\right]_0^{\pi}$\\

$ = \dfrac{2}{\pi}(1-(-1))$\\

$ = \dfrac{4}{\pi}$\\

$ a_n = \dfrac{1}{\pi}\displaystyle\int_{-\pi}^{\pi} f(x)\cos(nx)\,dx$\\

$ = \dfrac{1}{\pi}\displaystyle\int_0^{\pi} 2\sin x \cos(nx)\,dx$\\

$ = \dfrac{1}{\pi}\displaystyle\int_0^{\pi}
\left(\sin((1+n)x)+\sin((1-n)x)\right)\,dx$\\


Voor $n=1$:\\

$ a_1 = \dfrac{1}{\pi}\displaystyle\int_0^{\pi}\sin(2x)\,dx = 0$\\

Voor $n\neq1$:\\

$ a_n =
\dfrac{1}{\pi}
\left[
-\dfrac{\cos((1+n)x)}{1+n}
-\dfrac{\cos((1-n)x)}{1-n}
\right]_0^{\pi}$\\

$ =\dfrac{2(( -1)^n+1)}{\pi(1-n^2)}$\\

Dus:\\

$ a_n = 0 \text{ voor oneven } n$\\

$ a_{2k} = \dfrac{4}{\pi(1-4k^2)}$\\

$ b_n = \dfrac{1}{\pi}\displaystyle\int_{-\pi}^{\pi} f(x)\sin(nx)\,dx$\\

$ = \dfrac{1}{\pi}\displaystyle\int_0^{\pi} 2\sin x \sin(nx)\,dx$\\

$ = \dfrac{1}{\pi}\displaystyle\int_0^{\pi}
\left(\cos((1-n)x)-\cos((1+n)x)\right)\,dx$\\

Voor $n=1$:\\

$ b_1 = \dfrac{1}{\pi}\displaystyle\int_0^{\pi}2\sin^2x\,dx$\\

$ = \dfrac{1}{\pi}\displaystyle\int_0^{\pi}(1-\cos 2x)\,dx$\\

$ = \dfrac{1}{\pi}\left[x-\dfrac{\sin 2x}{2}\right]_0^{\pi}$\\

$ = 1$\\

Voor $n>1$:\\

$ b_n =
\dfrac{1}{\pi}
\left[
\dfrac{\sin((1-n)x)}{1-n}
-
\dfrac{\sin((1+n)x)}{1+n}
\right]_0^{\pi}$\\

$ = 0$\\

Fourierreeks:\\

$ f(x) \sim \dfrac{a_0}{2} + \displaystyle\sum_{n=1}^{\infty}
\left(a_n\cos(nx)+b_n\sin(nx)\right)$\\

$ f(x) \sim \dfrac{2}{\pi} + \sin x +
\displaystyle\sum_{k=1}^{\infty}
\dfrac{4}{\pi(1-4k^2)}\cos(2kx)$\\
