$ f_n : \mathbb{R}_0^+ \rightarrow \mathbb{R}  : x \rightarrow \dfrac{1}{nx^2} $\\

$ \lim\limits_{n \rightarrow \infty}\dfrac{1}{nx^2} = 0 $\\

$ \Rightarrow f : \mathbb{R}_0^+ \rightarrow \mathbb{R} : x \rightarrow 0 $ en $  f_n \stackrel{p}{\rightarrow} f $\\

$ \left|\left|f_n - f \right|\right|_{\infty} = \sup\limits_{x \in \mathbb{R}}\left|\dfrac{1}{nx^2} - 0 \right| = +\infty $\\

$ \Rightarrow \lim\limits_{n \rightarrow \infty}\left|\left|f_n - f \right|\right|_{\infty} = +\infty $\\

$ \Rightarrow f_n \stackrel{u}{\not\rightarrow} f $\\
