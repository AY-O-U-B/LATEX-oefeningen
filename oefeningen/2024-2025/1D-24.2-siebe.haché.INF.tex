Opgave: Los de volgende vergelijking op in $\mathbb{C}$\\

$8z^6 + 7iz^3 + 1 = 0$\\

Stel $w = z^3$. \\

$8w^2 + 7iw + 1 = 0$\\

$D = (7i)^2 - 4 \cdot 8 \cdot 1 = -81$\\

Resolvente:\\

$\left\{ \begin{array}{l} 
	x^2 - y^2 = -81\\ 
	
	2xy = 0 \Rightarrow x^2(-y^2) = 0 
\end{array} \right. $\\

$\lambda^2 + 81\lambda = 0 \Rightarrow \lambda_1 = 0, \lambda_2 = -81$\\

$x^2 = 0 \Rightarrow x = 0$ en $y^2 = 81 \Rightarrow y = 9$\\

$w_1 = \dfrac{-7i + 9i}{16} = \dfrac{1}{8}i$ en $w_2 = \dfrac{-7i - 9i}{16} = -i$\\

$\Rightarrow z^3 = \dfrac{1}{8}i$ en $z^3 = -i$\\


oplossingen:\\

voor $w_1$\\

$\left|\dfrac{1}{8}i\right| = \dfrac{1}{8}$\\

$\theta_1 = \arg(w_1) = \dfrac{\pi}{2}$\\

$z_k = \sqrt[3]{\dfrac{1}{8}} \left( \cos \dfrac{\dfrac{\pi}{2} + 2k\pi}{3} + i \sin \dfrac{\dfrac{\pi}{2} + 2k\pi}{3} \right)$\\


$k=0 \Rightarrow z_0 = \dfrac{1}{2} \left( \cos \dfrac{\pi}{6} + i \sin \dfrac{\pi}{6} \right) = \dfrac{\sqrt{3}}{4} + \dfrac{1}{4}i$\\


$k=1 \Rightarrow z_1 = \dfrac{1}{2} \left( \cos \dfrac{5\pi}{6} + i \sin \dfrac{5\pi}{6} \right) = -\dfrac{\sqrt{3}}{4} + \dfrac{1}{4}i$\\


$k=2 \Rightarrow z_2 = \dfrac{1}{2} \left( \cos \dfrac{9\pi}{6} + i \sin \dfrac{9\pi}{6} \right) = \dfrac{1}{2} (0 - i) = -\dfrac{1}{2}i$\\

voor $w_2$\\

$\left|-i\right| = 1$\\

$\theta_2 = \arg(w_2) = \dfrac{3\pi}{2}$ \\


$z_k = \sqrt[3]{1} \left( \cos \dfrac{\dfrac{3\pi}{2} + 2k\pi}{3} + i \sin \dfrac{\dfrac{3\pi}{2} + 2k\pi}{3} \right)$\\


$k=0 \Rightarrow z_3 = \cos \dfrac{\pi}{2} + i \sin \dfrac{\pi}{2} = i$\\


$k=1 \Rightarrow z_4 = \cos \dfrac{7\pi}{6} + i \sin \dfrac{7\pi}{6} = -\dfrac{\sqrt{3}}{2} - \dfrac{1}{2}i$\\


$k=2 \Rightarrow z_5 = \cos \dfrac{11\pi}{6} + i \sin \dfrac{11\pi}{6} = \dfrac{\sqrt{3}}{2} - \dfrac{1}{2}i$\\
