= -\displaystyle\int\sin^3t\cdot\sin tdt $\\

$ -\displaystyle\int\dfrac{\left(1-\cos 2t\right)^2}{4}dt = -\dfrac{1}{4}\displaystyle\int dt +\dfrac{1}{2}\displaystyle\int\cos 2tdt - \dfrac{1}{4}\displaystyle\int\cos^2tdt $\\

$ = -\dfrac{1}{4}t +\dfrac{1}{4}\sin 2t - \dfrac{1}{4}\displaystyle\int\cos^2tdt = -\dfrac{1}{4}t +\dfrac{1}{4}\sin 2t - \dfrac{1}{4}\displaystyle\int\dfrac{1+\cos 4t}{2}tdt $\\

$ = -\dfrac{1}{4}t +\dfrac{1}{4}\sin 2t - \dfrac{1}{8}t - \dfrac{1}{8}\displaystyle\int\cos 4tdt $\\

$ = -\dfrac{1}{4}t +\dfrac{1}{4}\sin 2t - \dfrac{1}{8}t - \dfrac{1}{32}\sin 4t + c $\\
